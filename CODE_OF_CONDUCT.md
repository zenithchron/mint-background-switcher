# Code of Conduct

## Our pledge

Contributors and maintainers pledge to make participation in this project a harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent, but accept corrections gracefully.
- Keep discussion focused on improving the project.
- Avoid personal attacks, harassment, or discriminatory language.

## Unacceptable behavior

- Harassment, intimidation, or threats.
- Derogatory comments related to identity or background.
- Publishing private information without explicit permission.
- Sustained disruption of project discussions.

## Enforcement

Maintainers may remove comments, close issues, or block participants who violate this code of conduct. Serious issues can be reported through GitHub repository moderation tools.
